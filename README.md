# EasyCluster

**Semi-automatic node discovery based on configuration**

## Installation

Add `easy_cluster` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:easy_cluster, "~> 0.1"}
  ]
end
```

## Usage

Try

```elixir
$ iex -S mix run.cluster
```
